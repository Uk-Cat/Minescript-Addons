This Mod is an addon to [Minescript](https://modrinth.com/mod/minescript). Minescript is a Minecraft mod that allows you to controll/interact with the game using python/pyjinn script created by [Maxuser](https://modrinth.com/user/maxuser).

This Minescript Addons Mod allows you to install community made script with only the link.

You will need [ModMenu](https://modrinth.com/mod/modmenu) as a required dependency.

### Commands:
```
/install [Github Repo]
```
```
/SetPath [Path Directory]
```

### **Features:**
- Ability to download script from just github link View the repository and Author in your browser at the click of a button.
- Download scripts with only commands
- Shortcuts in pause screen
- Copy to clipboard errors and Auto Copy to Clipboard

### **Planned features:**
- Automatically adding "from system.lib.minescript import *" at the start of every python file 
- Downloading multiple repos from a json text for easy transfer between profiles

The Mod is currently only made for 1.21.11 Fabric, other version may soon come however that is not guaranteed.
